JSessionHandlerInterface::regenerate

Regenerates ID that represents this storage.

Note regenerate+destroy should not clear the session data in memory only delete the session data from persistent storage.

    Signature

    public function regenerate([boolean $destroy = false, [integer $lifetime = null]] )

    Parameters

    $destroy — boolean
    Destroy session when regenerating?
    $lifetime — integer
    Sets the cookie lifetime for the session cookie. A null value will leave the system settings unchanged,
    0 sets the cookie to expire with browser session. Time is in seconds, and is not a Unix timestamp.

    Returns

    boolean
    True if session regenerated, false if error
