phpDocumentor\Reflection\DocBlock\TagFactory::registerTagHandler

Registers a handler for tags.

If you want to use your own tags then you can use this method to instruct the TagFactory to register the name of a tag with the FQCN of a 'Tag Handler'. The Tag handler should implement the interface (and thus the create method).
