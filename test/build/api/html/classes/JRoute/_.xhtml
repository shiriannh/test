JRoute::_

Translates an internal Joomla URL to a humanly readable URL.

    Signature

    public function _(string $url, [boolean $xhtml = true, [integer $ssl = null]] )

    Parameters

    $url — string
    Absolute or Relative URI to Joomla resource.
    $xhtml — boolean
    Replace & by &amp; for XML compliance.
    $ssl — integer
    Secure state for the resolved URI.
    0: (default) No change, use the protocol currently used in the request
    1: Make URI secure using global secure site URI.
    2: Make URI unsecure using the global unsecure site URI.

    Returns

    string
    The translated humanly readable URL.
