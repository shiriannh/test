JHtml::script

Write a `<script>` element to load a JavaScript file

    Signature

    public function script(string $file, [boolean $framework = false, [boolean $relative = false, [boolean $path_only = false, [boolean $detect_browser = true, [boolean $detect_debug = true]]]]] )

    Parameters

    $file — string
    path to file.
    $framework — boolean
    load the JS framework.
    $relative — boolean
    path to file is relative to /media folder.
    $path_only — boolean
    return the path to the file only.
    $detect_browser — boolean
    detect browser to include specific browser js files.
    $detect_debug — boolean
    detect debug to search for compressed files if debug is on.

    Returns

    mixed
    nothing if $path_only is false, null, path or array of path if specific js browser files were detected.
