PDepend\Source\AST\ASTClosure::isStatic

Returns whether this closure was defined as static or not.

This method will return <b>TRUE</b> when the closure was declared as followed: <code> $closure = static function( $e ) { return pow( $e, 2 ); } </code> And it will return <b>FALSE</b> when we declare the closure as usual: <code> $closure = function( $e ) { return pow( $e, 2 ); } </code>
