JTable::save

Method to provide a shortcut to binding, checking and storing a JTable instance to the database table.

The method will check a row in once the data has been stored and if an ordering filter is present will attempt to reorder the table rows based on the filter. The ordering filter is an instance property name. The rows that will be reordered are those whose value matches the JTable instance for the property specified.

    Signature

    public function save(array|object $src, [string $orderingFilter = '', [string $ignore = '']] )

    Parameters

    $src — object
    An associative array or object to bind to the JTable instance.


    $orderingFilter — string
    Filter for the order updating
    $ignore — string
    An optional array or space separated list of properties to ignore while binding.


    Returns

    boolean
    True on success.
