ezcConsoleOptionRule::__construct

Creates a new option rule.

Creates a new option rule. Per default the $values parameter is an empty array, which determines that the option may accept any value. To indicate that a option may only have certain values, place them inside tha $values array. For example to indicate an option may have the values 'a', 'b' and 'c' use: <code> $rule = new ezcConsoleOptionRule( $option, array( 'a', 'b', 'c' ) ); </code> If you want to allow only 1 specific value for an option, you do not need to wrap this into an array, when creating the rule. Simply use <code> $rule = new ezcConsoleOptionRule( $option, 'a' ); </code> to create a rule, that allows the desired option only to accept the value 'a'. The $ifSet parameter determines, if the rule is validated when its option is set or left out. If $ifSet is true, the rule is validated when the option is set. Otherwise the rule is validated if the option was not set by the user.
