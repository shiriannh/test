  • getDefaultValue() — This method will return the declared default value for this parameter.
  • getEndLine() — Returns the line number where the item declaration ends.
  • getName() — Returns the item name.
  • getPosition() — Returns the parameter position in the method/function signature.
  • getStartLine() — Returns the line number where the item declaration can be found.
  • isArray() — This method will return <b>true</b> when the parameter was declared with the array type hint, otherwise the it will return <b>false</b>.
  • isDefaultValueAvailable() — This method will return <b>true</b> when the parameter declaration contains a default value.
  • isOptional() — This method will return <b>true</b> when this parameter is optional and can be left blank on invocation.
  • isPassedByReference() — This method will return <b>true</b> when the parameter is passed by reference.
  • setDeclaringFunction() — Sets the parent function or method object.
  • setOptional() — This method can be used to mark a parameter optional. Note that a parameter is only optional when it has a default value an no following parameter has no default value.
  • setPosition() — Sets the parameter position in the method/function signature.
