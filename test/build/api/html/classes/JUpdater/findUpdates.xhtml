JUpdater::findUpdates

Finds the update for an extension. Any discovered updates are stored in the #__updates table.

    Signature

    public function findUpdates([integer $eid = 0, [integer $cacheTimeout = 0, [integer $minimum_stability = self::STABILITY_STABLE, [boolean $includeCurrent = false]]]] )

    Parameters

    $eid — integer
    Extension Identifier or list of Extension Identifiers; if zero use all
    sites
    $cacheTimeout — integer
    How many seconds to cache update information; if zero, force reload the
    update information
    $minimum_stability — integer
    Minimum stability for the updates; 0=dev, 1=alpha, 2=beta, 3=rc,
    4=stable
    $includeCurrent — boolean
    Should I include the current version in the results?

    Returns

    boolean
    True if there are updates
