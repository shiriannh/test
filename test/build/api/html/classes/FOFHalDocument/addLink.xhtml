FOFHalDocument::addLink

Add a link to the document

    Signature

    public function addLink(string $rel, FOFHalLink $link, [boolean $overwrite = true] )

    Parameters

    $rel — string
    The relation of the link to the document.
    See RFC 5988 http://tools.ietf.org/html/rfc5988#section-6.2.2 A document MUST always have
    a "self" link.
    $link — FOFHalLink
    The actual link object


    $overwrite — boolean
    When false and a link of $rel relation exists, an array of links is created. Otherwise the
    existing link is overwriten with the new one

    boolean
    True if the link was added to the collection
