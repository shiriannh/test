PDepend\Source\AST\ASTFieldDeclaration

This class represents a field or property declaration of a class.

<code>
// Simple field declaration
class Foo {
protected $foo;
}

// Field declaration with multiple properties
class Foo {
protected $foo = 23
$bar = 42,
$baz = null;
}
</code>
