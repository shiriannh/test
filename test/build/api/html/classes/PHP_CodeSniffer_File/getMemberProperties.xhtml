PHP_CodeSniffer_File::getMemberProperties

Returns the visibility and implementation properties of the class member variable found at the specified position in the stack.

The format of the array is: <code> array( 'scope' => 'public', // public private or protected 'is_static' => false, // true if the static keyword was found. ); </code>
