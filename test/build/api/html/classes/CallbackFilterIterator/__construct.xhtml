CallbackFilterIterator::__construct

Creates a filtered iterator using the callback to determine which items are accepted or rejected.

    Signature

    public function __construct(Iterator $iterator, callable $callback )

    Parameters

    $iterator — \Iterator
    The iterator to be filtered.


    $callback — object
    The callback, which should return TRUE to accept the current item
    or FALSE otherwise. May be any valid callable value.
    The callback should accept up to three arguments: the current item,
    the current key and the iterator, respectively.
    ``` php
    function my_callback($current, $key, $iterator)
    ```
