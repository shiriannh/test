JText::_

Translates a string into the current language.

Examples: `<script>alert(Joomla.JText._('<?php echo JText::_("JDEFAULT", array("script"=>true)); ?>'));</script>` will generate an alert message containing 'Default' `<?php echo JText::_("JDEFAULT"); ?>` will generate a 'Default' string

    Signature

    public function _(string $string, [boolean $jsSafe = false, [boolean $interpretBackSlashes = true, [boolean $script = false]]] )

    Parameters

    $string — string
    The string to translate.
    $jsSafe — boolean
    Boolean: Make the result javascript safe.
    $interpretBackSlashes — boolean
    To interpret backslashes (\\=\, \n=carriage return, \t=tabulation)
    $script — boolean
    To indicate that the string will be push in the javascript language store

    Returns

    string
    The translated string or the key is $script is true
