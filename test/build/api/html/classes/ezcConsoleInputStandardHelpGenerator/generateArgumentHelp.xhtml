ezcConsoleInputStandardHelpGenerator::generateArgumentHelp

Generates help information as a multi-dimensonal array for the given $argumentDefinition.

This method generates a tabular help information for the given $argumentDefinition in the following format: <code> <?php array( 0 => array( 0 => '<argument synopsis>', 1 => '<argument help text>' ), 1 => array( 0 => '<argument synopsis>', 1 => '<argument help text>' ), // ... ) ?> </code> The $long parameter defines if the long of short help text should be used.
