    Members

    public

    • $convertToInt — array
      Array of object fields to convert to integers before calculating SHA1 hash. Some values are stored differently when an item is created than when the item is changed and saved. This works around that issue.
    • $ignoreChanges — array
      Array of object fields to unset from the data object before calculating SHA1 hash. This allows us to detect a meaningful change in the database row using the hash. This can be read from the #__content_types content_history_options column.

    • __construct() — Constructor
    • deleteOldVersions() — Utility method to remove the oldest versions of an item, saving only the most recent versions.
    • getHashMatch() — Utility method to get a matching row based on the hash value and id columns.
    • getSha1() — Utility method to get the hash after removing selected values. This lets us detect changes other than modified date (which will change on every save).
    • store() — Overrides JTable::store to set modified hash, user id, and save date.
