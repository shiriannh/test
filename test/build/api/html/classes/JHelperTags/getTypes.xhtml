JHelperTags::getTypes

Method to get a list of types with associated data.

    Signature

    public function getTypes([string $arrayType = 'objectList', [array $selectTypes = null, [boolean $useAlias = true]]] )

    Parameters

    $arrayType — string
    Optionally specify that the returned list consist of objects, associative arrays, or arrays.
    Options are: rowList, assocList, and objectList
    $selectTypes — array
    Optional array of type ids to limit the results to. Often from a request.
    $useAlias — boolean
    If true, the alias is used to match, if false the type_id is used.

    Returns

    array
    Array of of types
