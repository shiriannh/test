  • createOrReturnCacheDirectory() — Returns the full qualified path for an existing cache directory or creates a new cache directory for the given cache entry key and returns the full qualified path for that cache directory.
  • ensureExists() — Ensures that the given <b>$cacheDir</b> really exists.
  • flush() — Flushes all contents below the configured cache root directory and writes a version file with the current software version.
  • flushDirectory() — Deletes all files and directories below the given <b>$cacheDir</b>.

  • getCacheDir() — Returns the cache root directory.
  • getVersionFile() — Returns the file name for the used version file.
  • isValidVersion() — Tests if the current software cache version is similar to the stored file system cache version.
  • readVersion() — Reads the stored cache version number from the cache root directory.
  • writeVersion() — Writes the current software cache version into a file in the cache root directory.
