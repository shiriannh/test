FinderIndexerParserHtml::removeBlocks

Method to remove blocks of text between a start and an end tag.

Each block removed is effectively replaced by a single space. Note: The start tag and the end tag must be different. Note: Blocks must not be nested. Note: This method will function correctly with multi-byte strings.
