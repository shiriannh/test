FinderTableMap::publish

Method to set the publishing state for a row or list of rows in the database table. The method respects checked out rows by other users and will attempt to checkin rows that it can after adjustments are made.

    Signature

    public function publish([mixed $pks = null, [integer $state = 1, [integer $userId = 0]]] )

    Parameters

    An array of primary key values to update. If not
    set the instance property value is used. [optional]
    $state — integer
    The publishing state. eg. [0 = unpublished, 1 = published] [optional]
    $userId — integer
    The user id of the user performing the operation. [optional]

    Returns

    boolean
