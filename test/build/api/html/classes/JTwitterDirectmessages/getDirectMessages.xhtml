JTwitterDirectmessages::getDirectMessages

Method to get the most recent direct messages sent to the authenticating user.

    Signature

    public function getDirectMessages([integer $since_id = 0, [integer $max_id = 0, [integer $count = 20, [boolean $entities = null, [boolean $skip_status = null]]]]] )

    Parameters

    $since_id — integer
    Returns results with an ID greater than (that is, more recent than) the specified ID.
    $max_id — integer
    Returns results with an ID less than (that is, older than) or equal to the specified ID.
    $count — integer
    Specifies the number of direct messages to try and retrieve, up to a maximum of 200.
    $entities — boolean
    When set to true, each tweet will include a node called "entities,". This node offers a variety of metadata
    about the tweet in a discreet structure, including: user_mentions, urls, and hashtags.
    $skip_status — boolean
    When set to either true, t or 1 statuses will not be included in the returned user objects.

    Returns

    array
    The decoded JSON response
