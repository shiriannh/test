JHtmlBehavior::modal

Add unobtrusive JavaScript support for modal links.

    Signature

    public function modal([string $selector = 'a.modal', [array $params = ]] )

    Parameters

    $selector — string
    The selector for which a modal behaviour is to be applied.
    $params — array
    An array of parameters for the modal behaviour.
    Options for the modal behaviour can be:
    - ajaxOptions
    - size
    - shadow
    - overlay
    - onOpen
    - onClose
    - onUpdate
    - onResize
    - onShow
    - onHide

    Returns

    void
