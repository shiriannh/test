MenusHelper::getMenuLinks

Get a list of menu links for one or all menus.

    Signature

    public function getMenuLinks([string $menuType = null, [integer $parentId = 0, [integer $mode = 0, [array $published = , [array $languages = ]]]]] )

    Parameters

    $menuType — string
    An option menu to filter the list on, otherwise all menu links are returned as a grouped array.
    $parentId — integer
    An optional parent ID to pivot results around.
    $mode — integer
    An optional mode. If parent ID is set and mode=2, the parent and children are excluded from the list.
    $published — array
    An optional array of states
    $languages — array
    Optional array of specify which languages we want to filter

    array
