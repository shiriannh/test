phpDocumentor\Reflection\TypeResolverTest::testResolvingCompoundTypesWithTwoArrays

This test asserts that the parameter order is correct.

When you pass two arrays separated by the compound operator (i.e. 'integer[]|string[]') then we always split the expression in its compound parts and then we parse the types with the array operators. If we were to switch the order around then 'integer[]|string[]' would read as an array of string or integer array; which is something other than what we intend.
