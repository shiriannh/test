JTableNested::_getTreeRepositionData

Method to get various data necessary to make room in the tree at a location for a node and its children. The returned data object includes conditions for SQL WHERE clauses for updating left and right id values to make room for the node as well as the new left and right ids for the node.

    Signature

    protected function _getTreeRepositionData( $referenceNode, integer $nodeWidth, [string $position = 'before'] )

    Parameters

    $referenceNode — object
    A node object with at least a 'lft' and 'rgt' with
    which to make room in the tree around for a new node.
    $nodeWidth — integer
    The width of the node for which to make room in the tree.
    $position — string
    The position relative to the reference node where the room
    should be made.

    Returns

    mixed
    Boolean false on failure or data object on success.
