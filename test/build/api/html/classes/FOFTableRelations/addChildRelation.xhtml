FOFTableRelations::addChildRelation

Add a 1:1 forward (child) relation. This adds relations for the getChild() method.

In other words: does a table HAVE ONE child Parent and child relations works the same way. We have them separated as it makes more sense for us humans to read code like $item->getParent() and $item->getChild() than $item->getRelatedObject('someRandomKeyName')
