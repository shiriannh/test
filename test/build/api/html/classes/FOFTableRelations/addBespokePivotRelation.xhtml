FOFTableRelations::addBespokePivotRelation

Add any bespoke relation which involves a pivot table.

    Signature

    protected function addBespokePivotRelation(string $relationType, string $itemName, string $tableClass, string $localKey, string $remoteKey, string $ourPivotKey, string $theirPivotKey, string $pivotTable, boolean $default )

    Parameters

    $relationType — string
    The type of the relationship (multiple)
    $itemName — string
    is how it will be known locally to the getRelatedItems method
    $tableClass — string
    if skipped it is defined automatically as ComponentnameTableItemname
    $localKey — string
    is the column containing our side of the FK relation, default: componentname_itemname_id
    $remoteKey — string
    is the remote table's FK column, default: componentname_itemname_id
    $ourPivotKey — string
    is the column containing our side of the FK relation in the pivot table, default: $localKey
    $theirPivotKey — string
    is the column containing the other table's side of the FK relation in the pivot table, default $remoteKey
    $pivotTable — string
    is the name of the glue (pivot) table, default: #__componentname_thisclassname_itemname with plural items (e.g. #__foobar_users_roles)
    $default — boolean
    is this the default children relationship?

    void
