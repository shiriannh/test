PHPUnit_Framework_MockObject_Matcher_StatelessInvocation::invoked

Registers the invocation $invocation in the object as being invoked.

This will only occur after matches() returns true which means the current invocation is the correct one. The matcher can store information from the invocation which can later be checked in verify(), or it can check the values directly and throw and exception if an expectation is not met. If the matcher is a stub it will also have a return value.
