PDepend\Source\AST\AbstractASTClassOrInterface::getParentClasses

Returns an array with all parents for the current class.

The returned array contains class instances for each parent of this class. They are ordered in the reverse inheritance order. This means that the direct parent of this class is the first element in the returned array and parent of this parent the second element and so on.
