PDepend\Util\Configuration::__get

Magic get method that is called by PHP's runtime engine whenever an undeclared object property is accessed through a read operation. This implementation of the magic get method checks if a configuration value for the given <b>$name</b> exists and returns the configuration value if a matching entry exists. Otherwise this method will throw an exception.
