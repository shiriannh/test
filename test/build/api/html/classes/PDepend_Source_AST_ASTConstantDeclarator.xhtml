PDepend\Source\AST\ASTConstantDeclarator

This class represents a single constant declarator within a constant definition.

<code>
class Foo
{
// --------
const BAR = 42;
// --------
}
</code>

Or in a comma separated constant defintion:

<code>
class Foo
{
// --------
const BAR = 42,
// --------

// --------------
const BAZ = 'Foobar',
// --------------

// ----------
const FOO = 3.14;
// ----------
}
</code>
