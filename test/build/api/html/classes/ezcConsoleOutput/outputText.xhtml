ezcConsoleOutput::outputText

Print text to the console.

Output a string to the console. If $format parameter is omitted, the default style is chosen. Style can either be a special style , a style name or 'default' to print with the default styling. The $format parameter defines the name of a format. Formats are defined through the $formats proprty, which contains format definitions in form of objects. The format influences the outer appearance of a message (e.g. color) as well as the target the message is printed to (e.g. STDERR).
