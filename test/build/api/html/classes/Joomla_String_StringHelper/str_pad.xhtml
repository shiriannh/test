Joomla\String\StringHelper::str_pad

UTF-8 aware alternative to str_pad()

Pad a string to a certain length with another string. $padStr may contain multi-byte characters.

    Signature

    public function str_pad(string $input, integer $length, [string $padStr = ' ', [integer $type = STR_PAD_RIGHT]] )

    Parameters

    $input — string
    The input string.
    $length — integer
    If the value is negative, less than, or equal to the length of the input string, no padding takes place.
    $padStr — string
    The string may be truncated if the number of padding characters can't be evenly divided by the string's length.
    $type — integer
    The type of padding to apply

    Returns

    string
