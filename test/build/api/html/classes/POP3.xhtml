\POP3

PHPMailer POP-Before-SMTP Authentication Class.

Specifically for PHPMailer to use for RFC1939 POP-before-SMTP authentication.
Does not support APOP.

  • Author: Richard Davey (original author) <rich@corephp.co.uk>
