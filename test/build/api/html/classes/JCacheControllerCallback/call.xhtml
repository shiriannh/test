JCacheControllerCallback::call

Executes a cacheable callback if not found in cache else returns cached output and result

Since arguments to this function are read with func_get_args you can pass any number of arguments to this method as long as the first argument passed is the callback definition. The callback definition can be in several forms: - Standard PHP Callback array see <https://secure.php.net/callback> [recommended] - Function name as a string eg. 'foo' for function foo() - Static method name as a string eg. 'MyClass::myMethod' for method myMethod() of class MyClass

    public function call()

    Returns

    mixed
    Result of the callback
